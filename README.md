# Milouv - Sistema de Repertório e Escala de Louvor

Sistema simples para organização de repertório e escalas do ministério de louvor da IMESC.

## Funcionalidades

* Visualizar escala do mês
* Ver repertório de cada culto
* Compartilhar repertório
* Banco de músicas do ministério
* Avisos e informações para a equipe

## Tecnologias

* HTML
* JavaScript
* Firebase Realtime Database
* Vercel (deploy)

## Estrutura do projeto

```
index.html        → interface principal
musicas.json      → banco local de músicas
youtube.png       → ícone YouTube
spotify.png       → ícone Spotify
cifra.png         → ícone Cifra
```

## Banco de dados (Firebase)

```
config
  mesAtivo

escalas
  2026-03
  2026-04

musicas

repertorios

informacoes
```

## Deploy

Hospedado automaticamente pela Vercel.
